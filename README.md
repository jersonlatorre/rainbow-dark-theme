# Dusk Dark

Dusk Dark — a beautiful dark theme for VS Code and related IDEs, featuring teal, cream, amber, orange, and red accents.

Inspired by the colors of dusk.

## Installation

1. Open **Extensions** sidebar panel in VS Code.
2. Search for `Dusk Dark`.
3. Click **Install**.
4. Click **Reload** to reload your editor.
5. Code > Preferences > Color Theme > **Dusk Dark**.

## Color Palette

- **Teal**: `#49D8EE`
- **Cream**: `#F5EFE7`
- **Amber**: `#FCDF5F`
- **Orange**: `#FFA24B`
- **Red**: `#FF5B65`
- **Muted Gray**: `#a39684`

**Enjoy!**
